# Camofox Browser — Home Assistant Add-on

[![GitHub Release](https://img.shields.io/github/release/mliradelc/camofox-ha-addon.svg)](https://github.com/mliradelc/camofox-ha-addon/releases)
[![License](https://img.shields.io/github/license/mliradelc/camofox-ha-addon.svg)](LICENSE)

Home Assistant add-on that packages **[Camofox Browser](https://github.com/jo-inc/camofox-browser)** — a stealth headless browser REST API server for AI agents, powered by the [Camoufox](https://github.com/daijro/camoufox) Firefox fork.

## What is Camofox Browser?

An anti-detect browser automation server. AI agents interact with it via a simple REST API to open tabs, take accessibility snapshots, click elements, type text, and navigate — all with C++-level fingerprint spoofing that bypasses Cloudflare, bot detection, and anti-scraping systems.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the ⋮ menu → **Repositories**
3. Add: `https://github.com/mliradelc/camofox-ha-addon`
4. Find **Camofox Browser** in the store and click **Install**
